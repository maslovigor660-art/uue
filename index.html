ℹ️Подсказка

Как включить Pages и делиться

  1. Включить Pages: Settings → Pages → Source: Deploy from a branch → Branch: main → Folder: /root → Save. Репозиторий должен быть публичным.
  2. Дождаться публикации: через 1–3 минуты в Settings → Pages появится URL. Твой адрес:
    https://maslovigor660-art.github.io/uue/
  3. Если 404: проверь, что index.html лежит в корне ветки main, Pages выбрано main /root, репо публичный. Нажми Save ещё раз и подожди минуту, потом открой URL (с завершающим слэшем).
  4. Экспорт: «Экспорт» → скачай wishlist.json → отправь файл.
  5. Импорт: «Импорт» → выбери wishlist.json → список загрузится.
  6. Ссылка для двоих: зайди на адрес выше, нажми «Ссылка для двоих», скопируй и открой на другом ПК — список сохранится там. При изменениях создавай новую ссылку.
