# Мини-вишлист

Одностраничный список желаний: поиск, фильтры по магазину, бюджетный слайдер, сортировки, вкладки «Активные/Купленные», отметка «куплено», приоритет, добавление/удаление, редактирование картинок и сброс к исходным товарам. Всё хранится в localStorage.

## Возможности
- Поиск по артикулу, магазину или заметке.
- Фильтр по магазину (WB / Ozon / Яндекс / Другое) и слайдер бюджета.
- Сортировка (цена ↑/↓, магазин), переключение вида (сетка/список).
- Вкладки «Активные» и «Купленные», счётчик видимых и их суммы.
- Отметка «куплено», приоритет, открытие/копирование ссылки, удаление.
- Добавление своих позиций с артикулом, ценой, магазином, заметкой и картинкой.
- Редактирование картинки в деталях, удаление купленных, сброс к исходному списку.

## Как открыть сайт
### Вариант 1: GitHub Pages
1. Репозиторий должен быть публичным, `index.html` лежит в корне ветки `main`.
2. В репозитории откройте **Settings → Pages**.
3. **Source**: выберите **Deploy from a branch**. **Branch**: `main`, **Folder**: `/root`. Нажмите **Save**.
4. Подождите 1–3 минуты. Откройте: **https://maslovigor660-art.github.io/uue/** (со слэшем на конце).
5. Если видите 404:
   - Ещё раз проверьте, что выбраны `main` и `/root`, нажмите **Save**.
   - Убедитесь, что `index.html` в корне ветки `main`.
   - Подождите минуту и обновите страницу.

### Вариант 2: локально
- Просто откройте `index.html` двойным щелчком в браузере **или** запустите через VS Code Live Server / любой статический сервер. Все данные будут сохраняться локально в вашем браузере.
