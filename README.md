# ВкусДоставка — Лендинг доставки еды

**ВкусДоставка** — это адаптивный одностраничный сайт для демонстрации меню доставки еды. Проект показывает навыки вёрстки и работы с макетом.

## 🍔 Что внутри
- Карточки товаров с названием, описанием и ценой
- Адаптивная вёрстка под мобильные устройства и десктопы
- Чистый и минималистичный дизайн
- Быстрая загрузка (лёгкий вес страницы)

## 🛠 Технологии
- HTML5
- CSS3 (Flexbox, Grid, медиа-запросы)
