# Мини-вишлист

Одностраничный список желаний с поиском, фильтрами, вкладками активных/купленных, бюджетным слайдером, приоритетами, отметкой «куплено», добавлением/удалением, импортом/экспортом JSON и «Ссылкой для двоих» (общая ссылка для синхронизации списка).

## Как открыть сайт (GitHub Pages)
1. Убедитесь, что репозиторий публичный.
2. Зайдите в **Settings → Pages**.
3. **Source**: выберите **Deploy from a branch**.
4. **Branch**: `main`, **Folder**: `/root` → **Save**.
5. Подождите 1–3 минуты, откройте: **https://maslovigor660-art.github.io/uue/** (с завершающим слэшем).
6. Если 404:
   - Проверьте, что `index.html` лежит в корне ветки `main`.
   - В Pages выбраны `main` и `/root`, нажмите **Save** ещё раз.
   - Подождите минуту и обновите URL.

## Импорт / экспорт
- **Экспорт**: на странице нажмите «Экспорт» → скачайте `wishlist.json`.
- **Импорт**: на странице нажмите «Импорт» → выберите `wishlist.json`.

## «Ссылка для двоих»
1. Откройте сайт по адресу Pages (не `file://`).
2. Нажмите «Ссылка для двоих» — ссылка скопируется.
3. Откройте её на другом компьютере: список загрузится и сохранится локально.

## Локально (без Pages)
- Откройте `index.html` напрямую в браузере **или** запустите через любой static server (например, VS Code Live Server). Общая ссылка корректно работает, если страница открыта по `http(s)://`.
